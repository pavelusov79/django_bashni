Cайт с новостройками

Стэк: python3.9.6, Django4.0. Полный список пакетов указан в файле requirements.txt

Новостройки по городам парсятся с с сайта нашдом.рф. 
Квартиры парсятся из трех парсеров отдельных сайтов ЖК.

На сервере в планировщике crontab запланировано задание ежедневно парсить квартиры.

После парсинга все данные записываются в БД mongo.
Затем с помощью скриптов в папке property/scripts/ выполняется перенос данных из монго в postgres. Добавлено логирование.

В базе чуть больше 1000 новостроек и примерно столько же квартир (добавлен поиск на подбор квартиры только по трем ЖК)

Возможности проекта:
1) CRUD операции с Users, News, Tags, BuildingObjects(и всех связанных моделей: фото, документов), City, Propery, Flats. В админке также предусмотрена возможность модерации комментариев пользователей, с уведомлением их по почте в случае отказа модерации.

Вход в админку: логин: bashni, пароль: bashnipswd

2) парсинг данных по новостройкам и квартирам
3) регистрация и авторизация пользователей
4) Поиск по сайту (по новостям, по тегам новостей, по названиям новостроек и квартирам)
5) на странице новости реализована возможнось поделиться новосью по почте или через соцсети. Возможность ставить лайки понравившимся новостям.Также можно оставить комментарий для авторизованных пользователей. Переход по тегам новостей.
6) фильтрация по новостройкам: по городам и различным параметрам. 
7) фильтрация по квартирам.
8) отправление запроса на подбор квартир в новостройке, а также отправление заявки на покупку квартиры.
9) отображение всей информации по отдельной новостройке
10) отображение всей информации по отдельной квартире.

Проект развернут с помощью gunicron + nginx
